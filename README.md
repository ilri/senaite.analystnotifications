# senaite.analystnotifications

**Senaite add-on to notify analysts when they are assigned a worksheet**

This repository contains a Senaite Lab Information Management System (LIMS) add-on that sends notifications to *Analyst* users when worksheets are assigned to them. It extends the Senaite platform by triggering notifications when assignment events occur in the workflow.

Senaite is an open-source LIMS built on Plone that manages laboratory samples and workflows, including creation of worksheets and assignment of analyses to analysts.
